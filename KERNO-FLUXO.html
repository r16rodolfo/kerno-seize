Quando migrar para o fluxo BMAD completo?

Se o projeto cresceu além de 3 features core, atraiu usuários pagantes, ou você está começando a perder o controle do que foi construído → é hora de rodar a Auditoria BMAD e considerar adicionar planejamento retroativo com os agentes. Não é retrabalho — é evolução.

Terminei o Retrofit — e agora?

O Retrofit entrega um projeto seguro e documentado. Mas antes de escalar (mais usuários, mais features, mais desenvolvedores), passe pela Auditoria BMAD. Os agentes vão revisar tudo que o Lovable gerou e apontar o que precisa ser melhorado — com prioridade clara e plano de ação.

→ Fase 4.5: Rodar Auditoria BMAD ou → Ir direto para o Monitoramento contínuo
